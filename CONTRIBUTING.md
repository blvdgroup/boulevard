# Contributing to Boulevard

This document outlines the steps to setup boulevard to begin hacking on it.

## Table of Contents

- [Initial Setup](#initial-setup)
- [Building](#building)
- [Linting](#linting)

## Initial Setup

```shell
git clone https://github.com/blvdgroup/boulevard.git
cd boulevard
yarn
lerna bootstrap
```

## Building

```shell
lerna run build
```

## Linting

```shell
lerna run lint
```
